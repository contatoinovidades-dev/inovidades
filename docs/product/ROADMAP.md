# Roadmap do Produto — Inovidades

## Fase 1 — MVP (Atual)
- Site estático publicado
- Estado inicial: Pernambuco
- Segmentos prioritários ativos
- SEO básico e estrutura para AdSense

## Fase 2 — Expansão Geográfica
- Replicação para outros estados
- Padronização de templates
- Crescimento orgânico via indexação

## Fase 3 — Automação
- Fontes automatizadas de dados
- Scripts de geração de páginas
- Atualização periódica de oportunidades

## Fase 4 — Evolução do Produto
- Filtros avançados
- Alertas
- Possível área dinâmica (sem quebrar SEO)

O roadmap é intencionalmente flexível e orientado a aprendizado.
